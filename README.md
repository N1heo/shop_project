## Java shop project
A simple - **terminal** application, which lets you control a shop.

App has 3 roles:
* Director
* Worker
* Delivery

Each role has it's own set of commands.

Project is made using **Java(jdk-17.0.4.1+1)** and **PostgreSQL13.**

Run file `src/App.java` to start the program.
